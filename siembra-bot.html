Arquitectura de Agentes

El sistema utiliza un enfoque modular para procesar el lenguaje natural y conectarlo con datos duros.

LLM (Large Language Model)

Motor central (Llama 3 / GPT) para interpretar la intención del agricultor y generar respuestas naturales.

Agente RAG

Recuperación de información técnica desde manuales de CONAGUA y SIAP para dar respuestas fundamentadas.

API Connector

Enlace en tiempo real con la base de datos de Siembra Link para leer el estado actual del campo.
