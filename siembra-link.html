Red de sensores IoT para monitoreo micro-climático en tiempo real.

Especificaciones Técnicas

Dispositivo autónomo basado en Arduino diseñado para operar en campo abierto. Recopila variables críticas para alimentar el modelo predictivo.

Lista de Materiales

  • Microcontrolador: ESP32 / Arduino Uno (Bajo consumo)
  • Sensor DHT22: Temperatura y Humedad relativa
  • Pluviómetro Digital: Medición de precipitación pluvial
  • Sensor Capacitivo: Humedad del suelo (resistente a corrosión)
  • Módulo LoRa/WiFi: Transmisión de datos de largo alcance
  • Energía: Panel solar 5V + Batería LiPo 2000mAh
