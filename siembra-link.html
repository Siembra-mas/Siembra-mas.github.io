Red de sensores IoT para monitoreo micro-climático en tiempo real.

El SiembraLink es el módulo de hardware de Internet de las Cosas (IoT) diseñado como la fuente sensorial de la arquitectura Siembra+. Su función es la adquisición de datos en tiempo real directamente desde el campo de cultivo, permitiendo digitalizar variables ambientales críticas para alimentar los modelos predictivos y mantener informado al usuario sobre el estado físico de su siembra.

Especificaciones Técnicas

Dispositivo autónomo basado en Arduino diseñado para operar en campo abierto. Recopila variables críticas para alimentar el modelo predictivo.

Lista de Materiales

  • Microcontrolador: ESP32 / Arduino Uno (Bajo consumo)
  • Sensor DHT22: Temperatura y Humedad relativa
  • Pluviómetro Digital: Medición de precipitación pluvial
  • Sensor Capacitivo: Humedad del suelo (resistente a corrosión)
  • Módulo LoRa/WiFi: Transmisión de datos de largo alcance
  • Energía: Panel solar 5V + Batería LiPo 2000mAh

Esquema de conexión de sensores al nodo central.

Arquitectura del Sistema

Metodología de funcionamiento y procesamiento de datos en el entorno físico.

Arquitectura

Sistema embebido modular centrado en la eficiencia energética. Utiliza microcontroladores (Arduino/ESP32) y programación en C++ para gestionar sensores de Humedad de Suelo, DHT11 y Lluvia.

Alcance Operativo

Opera físicamente en el terreno de cultivo, sirviendo como puente vital entre el mundo físico (tierra/clima) y la plataforma digital, con despliegue en sitio y sincronización inalámbrica.

Adquisición de Datos

Transforma señales analógicas y digitales en métricas estandarizadas. Realiza lectura cíclica, normalización de valores (0-100%) y calibración para reducir ruido.

Conectividad

Garantiza la transmisión continua mediante protocolos web (HTTP/HTTPS), enviando paquetes de datos en formato JSON hacia la API del servidor para su almacenamiento.

Funcionalidad

Monitoreo activo para detectar anomalías (sequía, exceso de lluvia). Los datos actualizan tableros en tiempo real y sirven de input para las recomendaciones del Asistente.

Tecnología accesible para el campo mexicano. Democratizando la agricultura de precisión con IoT e Inteligencia Artificial.
